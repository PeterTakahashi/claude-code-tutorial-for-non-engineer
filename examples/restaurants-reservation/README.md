# 予約サイト：コピペ用プロンプト集

スライド「作る順番（STEP 0〜9）」で作る **お店の予約サイト** を、Claude Code に順番に頼んでいくためのプロンプト集です。

- 本体は [`prompts.md`](./prompts.md)。**上から1つずつ**コピペして送ってください。
- 1ステップごとに「動く状態」にしてから次へ進みます（いきなり全部は頼まない）。
- 出てきた変更は **差分（diff）を見て Accept** 。おかしければ「さっきのをこう直して」と言い直せばOK。

## 前提（始める前に）
- **Docker Desktop を起動しておく**（DBなどをコンテナで動かします）。
- Claude Desktop の **Code タブ → Local** で、このフォルダ（または新しい空フォルダ）を作業場所に選ぶ。
- ngrok の authtoken を取得済み（最後の公開ステップで使います）。

## 使う技術（Claudeにこの構成で作らせます）
- バックエンド：**Go + Gin + GORM** / DB：**PostgreSQL**
- フロント：**Next.js（React・TypeScript）+ Tailwind CSS + shadcn/ui**、表は **TanStack Table**、入力チェックは **Zod**
- 型の自動生成：**openapi-typescript**（バック↔フロントの型を一致させる）
- ローカル環境：**Docker（docker compose）**、ファイル保管が必要なら **MinIO**（S3互換）
